# Mô tả

SKILL brainstorming dùng trước mọi công việc sáng tạo hoặc thay đổi sản phẩm/code: tạo tính năng, xây component, thêm chức năng, sửa hành vi, thiết kế UI/UX, hoặc lập spec.

Mục đích là làm rõ ý tưởng trước khi triển khai: hiểu bối cảnh dự án, hỏi yêu cầu, phân tích hướng làm, đề xuất thiết kế, viết spec, cho user duyệt rồi mới chuyển sang lập kế hoạch implementation.
