Core cc- Flow

Các skill dùng trực tiếp trong chuỗi BA/PM/PO từ discovery đến handoff kỹ thuật.

01 / Discovery

Product Discovery Writer

Khám phá vấn đề, cơ hội, stakeholder, feasibility và rủi ro trước khi quyết định viết PRD.

  • Problem framing
  • Stakeholder map
  • Go/No-Go
  • PRD kickoff
.claude/skills/cc-pd-writer ↗
02 / PRD

PRD Writer

Viết Product Requirements Document ở góc nhìn nghiệp vụ, kết quả mong đợi và phạm vi sản phẩm.

  • FR / NFR
  • Scope
  • Release plan
  • RTM
.claude/skills/cc-prd-writer ↗
03 / Agile

User Story & Acceptance Criteria

Chia nhỏ yêu cầu thành User Story đạt INVEST và Acceptance Criteria dạng Given-When-Then.

  • INVEST
  • Happy path
  • Edge case
  • Error path
.claude/skills/cc-user-story-acceptance-criteria-writer ↗
04 / Interaction

Use Case Writer

Mô tả tương tác actor-system theo template 13 trường Karl Wiegers / IIBA và kiểm tra phạm vi user-goal.

  • Primary actor
  • Normal course
  • Alternative
  • Exceptions
.claude/skills/cc-use-case-writer ↗
05 / Technical

SRS Writer

Chuyển PRD/Use Case thành SRS: interface, contract, data model, processing rules và traceability.

  • API contract
  • Data model
  • NFR
  • ADR scoped
.claude/skills/cc-srs-writer ↗
06 / QA

Test Case Writer

Chuyển PRD, User Story, AC, Use Case hoặc SRS thành test cases có traceability và coverage matrix.

  • Test data
  • Steps
  • Expected result
  • Coverage
.claude/skills/cc-test-case-writer ↗

Ranh giới bàn giao

Các `cc-*` skills hoạt động tốt nhất khi mỗi tài liệu giữ đúng vai trò trong chuỗi.

BA / PM / PO

  • Discovery trả lời có nên đầu tư và cần xác thực giả định nào.
  • PRD mô tả hành vi sản phẩm, phạm vi, quy tắc nghiệp vụ và kết quả mong đợi.
  • User Story + AC chia nhỏ yêu cầu để estimate, plan và test theo Gherkin.

Dev / QA / Architect

  • Use Case làm rõ tương tác actor-system ở user-goal level.
  • SRS chuyển yêu cầu nguồn thành interface, contract, data model và NFR đo được.
  • Test Case Writer tạo test data, steps, expected results và coverage matrix để QA execute.
